# A4 QR Code Sheet — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Generate a printable A4 sheet (HTML → browser print to PDF) with a QR code for `youyou.mprnl.fr`, styled in the "poetic postcard" design from the spec.

**Architecture:** A standalone Node.js script generates `a4-qrcode.html` — a self-contained HTML file with embedded QR code (SVG data URL), exact design spec CSS (dark background, gold accents, centered layout), and print-optimized CSS for A4. No React/Vite involved; the HTML file opens directly in a browser for print-to-PDF.

**Tech Stack:** Node.js + TypeScript, `qrcode` npm package, vanilla HTML/CSS

```javascript
import QRCode from "qrcode";
import fs from "fs";
import path from "path";
import { fileURLToPath } from "url";

const __dirname = path.dirname(fileURLToPath(import.meta.url));

async function generate() {
  const url = "https://youyou.mprnl.fr";

  const qrSvg = await QRCode.toString(url, {
    type: "svg",
    color: { dark: "#D4A853", light: "#1E1A24" },
    width: 256,
    margin: 2,
  });

  const extractSvgInner = (svg: string): string => {
    const match = svg.match(/<svg[^>]*>([\s\S]*)<\/svg>/);
    return match ? match[1] : svg;
  };

  const qrInner = extractSvgInner(qrSvg);

  const html = `<!doctype html>
<html lang="fr">
<head>
<meta charset="UTF-8" />
<title>Pour toi — Youna</title>
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<link rel="preconnect" href="https://fonts.googleapis.com" />
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
<link href="https://fonts.googleapis.com/css2?family=Caveat:wght@400;600;700&family=Inter:wght@300;400;500&family=Playfair+Display:ital,wght@0,400;0,600;0,700;1,400;1,600&display=swap" rel="stylesheet" />
<style>
  @page {
    size: A4 portrait;
    margin: 0;
  }

  * {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
  }

  html, body {
    width: 210mm;
    height: 297mm;
    overflow: hidden;
    background: #1E1A24;
    font-family: 'Inter', 'system-ui', sans-serif;
    -webkit-print-color-adjust: exact;
    print-color-adjust: exact;
  }

  .sheet {
    width: 210mm;
    height: 297mm;
    background: #1E1A24;
    display: flex;
    align-items: center;
    justify-content: center;
    position: relative;
    overflow: hidden;
  }

  .sheet::before {
    content: '';
    position: absolute;
    top: -80px;
    left: 50%;
    transform: translateX(-50%);
    width: 280px;
    height: 280px;
    border-radius: 50%;
    background: radial-gradient(circle, rgba(212,168,83,0.07) 0%, transparent 70%);
    pointer-events: none;
  }

  .sheet::after {
    content: '';
    position: absolute;
    bottom: -40px;
    left: 50%;
    transform: translateX(-50%);
    width: 200px;
    height: 200px;
    border-radius: 50%;
    background: radial-gradient(circle, rgba(123,107,138,0.06) 0%, transparent 70%);
    pointer-events: none;
  }

  .content {
    position: relative;
    z-index: 1;
    text-align: center;
    display: flex;
    flex-direction: column;
    align-items: center;
  }

  .line-top {
    width: 60px;
    height: 1px;
    background: linear-gradient(90deg, transparent, rgba(212,168,83,0.5), transparent);
    margin-bottom: 30px;
  }

  .label {
    font-family: 'Playfair Display', Georgia, serif;
    font-style: italic;
    color: #D4A853;
    font-size: 16px;
    letter-spacing: 3px;
    opacity: 0.85;
    margin-bottom: 8px;
  }

  .name {
    font-family: 'Playfair Display', Georgia, serif;
    color: #FFF8EC;
    font-size: 52px;
    font-weight: 600;
    line-height: 1;
    letter-spacing: -1px;
    margin-bottom: 12px;
  }

  .divider {
    width: 50px;
    height: 1px;
    background: linear-gradient(90deg, transparent, #D4A853, transparent);
    margin: 0 auto 20px;
  }

  .subtitle {
    font-family: 'Caveat', cursive;
    color: #E8D5B7;
    font-size: 20px;
    opacity: 0.65;
    line-height: 1.4;
    margin-bottom: 40px;
  }

  .qr-wrapper {
    border: 2px solid rgba(212,168,83,0.4);
    padding: 8px;
    border-radius: 16px;
    width: 128px;
    height: 128px;
    background: rgba(255,248,236,0.04);
    box-shadow: 0 0 35px rgba(212,168,83,0.12), 0 0 60px rgba(212,168,83,0.05);
  }

  .qr-wrapper svg {
    width: 100%;
    height: 100%;
    display: block;
  }

  .url {
    font-family: 'Inter', 'system-ui', sans-serif;
    color: #7B6B8A;
    font-size: 10px;
    letter-spacing: 2px;
    text-transform: uppercase;
    opacity: 0.7;
    margin-top: 14px;
  }

    width: 60px;
    height: 1px;
    background: linear-gradient(90deg, transparent, rgba(212,168,83,0.5), transparent);
    margin-top: 40px;

    pointer-events: none;

      display: flex;

      background: #0d0a12;
    }

      box-shadow: 0 0 60px rgba(0,0,0,0.5);
      border-radius: 4px;
    }
  }
</style>
</head>
<body>
  <div class="sheet">
    <div class="star" style="top:15%;left:15%;width:2px;height:2px;background:rgba(212,168,83,0.3);"></div>
    <div class="star" style="top:20%;right:18%;width:1px;height:1px;background:rgba(212,168,83,0.2);"></div>
    <div class="star" style="top:65%;left:12%;width:1.5px;height:1.5px;background:rgba(212,168,83,0.25);"></div>
    <div class="star" style="top:55%;right:14%;width:1px;height:1px;background:rgba(255,248,236,0.15);"></div>
    <div class="star" style="bottom:25%;left:20%;width:1.5px;height:1.5px;background:rgba(212,168,83,0.2);"></div>
    <div class="star" style="top:35%;right:22%;width:1px;height:1px;background:rgba(255,248,236,0.12);"></div>

    <div class="content">
      <div class="line-top"></div>
      <p class="label">pour toi</p>
      <h1 class="name">Youna</h1>
      <div class="divider"></div>
      <p class="subtitle">Scanne et laisse-toi porter</p>
      <div class="qr-wrapper">
        <svg viewBox="0 0 256 256" xmlns="http://www.w3.org/2000/svg">
          ${qrInner}
        </svg>
      </div>
      <p class="url">youyou.mprnl.fr</p>
      <div class="line-bottom"></div>
    </div>
  </div>
</body>
</html>`;

  const outPath = path.resolve(__dirname, "..", "a4-qrcode.html");
  fs.writeFileSync(outPath, html);
  console.log(`Generated: ${outPath}`);
}

generate().catch(console.error);
```

### Task 4: Print to PDF

- [ ] **Step 1: Open the HTML file in browser**

```bash
open a4-qrcode.html  # macOS
# OR
xdg-open a4-qrcode.html  # Linux
```

- [ ] **Step 2: Print to PDF**

In the browser:
1. `Ctrl+P` (or `Cmd+P`)
2. Set destination to "Save as PDF"
3. Paper size: A4
4. Margins: None
5. Check "Background graphics" is ON
6. Save

- [ ] **Step 3: Verify the PDF**

Open the PDF and verify:
- QR code is visible and centered
- Colors are correct (dark background, gold text)
- Layout matches the design spec
- QR code is scannable (test with phone)
